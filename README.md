# Book-Recommendation-System
Book Recommendation System built with Python and Machine Learning to provide personalized book suggestions using collaborative filtering and content-based recommendation techniques. The project includes data preprocessing, feature engineering, similarity computation, and an interactive Streamlit web application.It demonstrates practical implementation of recommendation algorithms, data analysis, and model deployment using Pandas, NumPy, Scikit-learn, and Pickle.

# 📚 Book Recommendation System

A Machine Learning-based Book Recommendation System that suggests books based on user preferences using collaborative filtering and content-based recommendation techniques.

## 🚀 Features

- Personalized book recommendations
- Content-Based & Collaborative Filtering
- Fast similarity search
- Interactive Streamlit web interface
- Clean and user-friendly UI

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle
- Streamlit
- Jupyter Notebook

## 📂 Project Structure

```
Book-Recommendation-System/
```

## 📊 Workflow

1. Load and preprocess the dataset.
2. Clean and transform book information.
3. Build recommendation models.
4. Calculate similarity scores.
5. Save trained models using Pickle.
6. Deploy the application with Streamlit.

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
